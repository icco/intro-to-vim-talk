# How to Install?
 
 * Ubuntu: `sudo apt-get install vim-full`
 * Fedora: `sudo yum install vim`
 * Mac: <http://code.google.com/p/macvim/>
 * Windows: <http://www.vim.org/download.php#pc>
 * CSC Machines: `vim`
